  • LINGUAGENS PRESENTES NOS APP’s

    Você já se perguntou como foram programadas e qual linguagem utilizaram?

    Facebook

    O Facebook foi desenvolvido, em 2004, com a linguagem PHP. Em 2011, com seu grande crescimento optaram pela utilização do Hip Hop for PHP, um transpilador fonte a fonte. Basicamente, esse transpilador "traduz" os códigos de PHP para C++.

    Em 2014, os desenvolvedores criaram o Hack, uma nova linguagem de programação para o HHVM (HipHop Virtual Machine). Essa nova linguagem tem um melhor desempenho e facilita visualmente os erros no código.

    Atualmente, o Front-End é desenvolvido em JavaScript. O Back-End em Hack. Outras linguagens como Python, C++, Java, Erlang, D, XHP e Haskell estão presentes em seu código.

    Twitter

    Como a maioria dos websites, o JavaScript é utilizado em seu front-end. No entanto, a estrutura Back-End era desenvolvido em Ruby e com seu banco de dados em MySQL.

    Contudo, os problemas de performance fizeram a rede social migrar seu Back-End para linguagem Scala. Por fim, também utilizaram um servidor Java, chamado Blender, para aprimorar sua performance em Search Stack.

    Instagram

    A plataforma utiliza serviços de:

    - Armazenamento da Amazon;

    - Sistema de gerenciamento de banco de dados com o PostgreSQL;

    - Nginx como web Server;

    - Python, como Data Science para visualização e processamentos de algoritmos de Inteligência Artificial;

    - Django como framework, uma ferramenta que auxilia o desenvolvedor a programar uma linguagem, no caso Python.

    WhatsApp

    Desde 2009 em sua criação, a plataforma foi desenvolvida somente para mobile. Porém, só em 2015 ganhou uma versão para desktop, desenvolvida em PHP.

    A linguagem de programação adotada é a Erlang. Uma linguagem enxuta, mas que permite aos programadores atualizarem ou corrigir bugs sem que o serviço pare de funcionar para milhões de usuários.

    Youtube

    Youtube, a maior plataforma de vídeos do mundo!

    Desenvolvida com linguagens PHP, Python, Actionscript e JavaScript e até em C++.

    O impressionante é sua infraestrutura elaborada e bem complexa. Por exemplo, a plataforma conta com 14 Data Centers do Google espalhadas pelo mundo.

    LinkedIn

    A maior rede social para profissionais do mundo.

    O LinkedIn apresenta uma arquitetura cheia de tecnologia, desenvolvida em Java e Scala. No banco de dados é utilizado Oracle e o MySQL.

  • LINGUAGEM DE PROGRAMAÇÃO DESENVOLVIDA POR UM BRASILEIRO

    José Valim, criador da linguagem de programação brasileira - Exilir - pela empresa de consultoria de software Plataformatec. O programador de Porto Alegre nascido no ano de 1986 é formado como Engenheiro pela Escola Politécnica da Universidade de São Paulo e possui mestrado em Ciência da Computação pela Politécnica de Torino, na Itália.

    Exilir é uma linguagem de programação funcional, distribuída, orientada a processos e concorrente. Linguagem open source, executada na Erlang VM (Máquina Virtual Erlang). Seu objetivo é ter uma programação produtiva para aplicações, solução de problemas e manutenção; tendo de diversos recursos e ferramentas de outras linguagens.

    A Honeypot fez um pequeno documentário falando sobre o Exilir.

  • LINGUAGENS DE PROGRAMAÇÃO EM ALTA

    Java

    Presente desde 1991, Java é uma das linguagens de programação mais utilizadas no mundo. Graças a sua capacidade de adaptar seu programa e sua utilização em tarefas do dia-a-dia.

    Utilizado na criação de games, aplicativos mobile. Tem como vantagem a sua retro compatibilidade, ou seja, versões antigas ainda podem ser entendidas pelas versões atuais.

    C++

    O C++, assim como o C# tem bases próximas da linguagem C. Criada em 1979, se comporta igual ao Java, linguagem de programação de uso geral. Muito vigente na programação de games por causa da alta performance, desempenho e flexibilidade.

    JavaScript

    A linguagem de programação que vem conquistando milhares de desenvolvedores/programadores de qualquer área do TI.

    Com JavaScript podemos criar interatividade com o usuário nas aplicações web, e está presente em quase todas as aplicações e plataformas.

    Por ser de fácil entendimento, muitos acreditam que seja a melhor linguagem para um iniciante adquirir familiaridade com a programação.

    Python

    Sendo uma linguagem de programação bem versátil e de leitura fácil, Python permite a digitação de códigos mais "clean", ou seja, com menos linhas. Lançado em 1989, Python possui software bases de Open Source e é muito utilizado em melhorias para a redes e Machine Learning.

    PHP

    PHP - Hypertext Preprocessor - é uma linguagem de script bastante usado para o desenvolvimento web. Pode ser embutida dentro de um HTML e tem integração entre o banco de dados MySQL.

    Exemplos de sites: Facebook e Yahoo.

    Ruby

    As sintaxes de leitura fácil é um ponto forte. Ruby tem a desvantagem de sua linguagem não ser escalonada. O que gera erro no código conforme os números de usuário aumentam.

    Airbnb utiliza linguagem Ruby.

    Swift

    A Swift é a atual linguagem utilizada pela Apple para desenvolver o iOS, MacOS, WatchOS, tvOS. Pois tem compatibilidade com a Objective-C.

    Sua popularidade cresce tanto, desde sua criação em 2014, que atualmente existem faculdades que oferecem o curso em sua grade curricular, por exemplo a Universidade Técnica de Munique e a Universidade de Stanford.

    Golang (Go)

    A Google criou uma linguagem de programação Open Source, influenciada pelas linguagens C e Pascal.

    A linguagem ganha credibilidade ao fazer parte de algumas plataformas mundialmente conhecidas, como:

    - Netflix;

    - SoundCloud;

    - Twitch;

    - Uber.

    Kotlin

    Assim como a Swift, a Kotlin foi desenvolvida para apoiar os programadores no desenvolvimento dos softwares e em aplicações Android.

    Vale lembrar que essa linguagem de programação é compatível com Java.

    Alguns exemplos são:

    - Pinterest;

    - Evernote;

    - Uber e etc.
